  AdSense "최상위 도메인" 오류 해결 가이드
═══════════════════════════════════════════════════════════════

❌ 문제: "URL은 올바른 최상위 도메인이어야 합니다"
✅ 해결: 최상위 도메인만 입력하면 됩니다!

───────────────────────────────────────────────────────────────

📌 상황별 해결 방법
───────────────────────────────────────────────────────────────

1️⃣ GitHub Pages를 사용하는 경우
───────────────────────────────────────────────────────────────
❌ 잘못된 입력:
   https://guitbe.github.io/salary-calculator
   (저장소 이름 포함)

✅ 올바른 입력:
   https://guitbe.github.io
   (저장소 이름 제외, 최상위 도메인만!)

💡 AdSense가 "입력하시려는 도메인이 http://guitbe.github.io입니까?" 
   라고 물어보면 → "예" 또는 "네" 클릭!

───────────────────────────────────────────────────────────────

2️⃣ Netlify를 사용하는 경우
───────────────────────────────────────────────────────────────
✅ 올바른 입력:
   https://사이트이름.netlify.app
   (예: https://salary-calculator.netlify.app)

❌ 잘못된 입력:
   https://사이트이름.netlify.app/경로
   (하위 경로 포함)

───────────────────────────────────────────────────────────────

3️⃣ Vercel을 사용하는 경우
───────────────────────────────────────────────────────────────
✅ 올바른 입력:
   https://사이트이름.vercel.app
   (예: https://salary-calculator.vercel.app)

───────────────────────────────────────────────────────────────

📌 최상위 도메인이란?
───────────────────────────────────────────────────────────────
최상위 도메인 = 루트 도메인 = 메인 도메인

예시:
✅ https://guitbe.github.io          (최상위 도메인)
❌ https://guitbe.github.io/calculator (하위 경로 포함)
❌ https://guitbe.github.io/salary     (하위 경로 포함)

───────────────────────────────────────────────────────────────

📌 GitHub Pages 특별 주의사항
───────────────────────────────────────────────────────────────
GitHub Pages는 두 가지 방식이 있습니다:

방식 1: 사용자/조직 페이지 (최상위 도메인)
   - 주소: https://사용자명.github.io
   - 저장소 이름: 반드시 "사용자명.github.io"여야 함
   - AdSense 신청 가능 ✅

방식 2: 프로젝트 페이지 (하위 경로)
   - 주소: https://사용자명.github.io/저장소이름
   - 저장소 이름: 아무거나 가능
   - AdSense 신청 불가 ❌ (최상위 도메인이 아니므로)

💡 해결책:
   1. 저장소 이름을 "사용자명.github.io"로 변경
   2. 또는 Netlify/Vercel 사용 (더 쉬움)

───────────────────────────────────────────────────────────────

📌 현재 상황 확인 방법
───────────────────────────────────────────────────────────────
1. 웹사이트 주소 확인
   - 주소에 "/저장소이름" 같은 경로가 있나요?
   - 있다면 → 하위 경로 포함 (AdSense 불가)
   - 없다면 → 최상위 도메인 (AdSense 가능)

2. 예시:
   ❌ https://guitbe.github.io/salary-calculator
      → 하위 경로 포함, AdSense 불가
   
   ✅ https://guitbe.github.io
      → 최상위 도메인, AdSense 가능

───────────────────────────────────────────────────────────────

📌 해결 방법 3가지
───────────────────────────────────────────────────────────────

방법 1: GitHub 저장소 이름 변경 (추천)
───────────────────────────────────────────────────────────────
1. GitHub 저장소 Settings → General
2. 저장소 이름을 "guitbe.github.io"로 변경
3. Settings → Pages에서 다시 활성화
4. 주소가 https://guitbe.github.io 로 변경됨
5. AdSense에 새 주소 입력

───────────────────────────────────────────────────────────────

방법 2: Netlify 사용 (가장 쉬움!)
───────────────────────────────────────────────────────────────
1. Netlify.com 가입
2. 파일 업로드
3. 자동으로 최상위 도메인 주소 발급
   (예: https://salary-calculator.netlify.app)
4. AdSense에 이 주소 입력

───────────────────────────────────────────────────────────────

방법 3: 사용자 도메인 연결 (고급)
───────────────────────────────────────────────────────────────
1. 도메인 구매 (예: salary-calculator.com)
2. GitHub Pages에 도메인 연결
3. AdSense에 도메인 주소 입력

───────────────────────────────────────────────────────────────

📌 AdSense 신청 시 주의사항
───────────────────────────────────────────────────────────────
1. 최상위 도메인만 입력
   ✅ https://guitbe.github.io
   ❌ https://guitbe.github.io/salary-calculator

2. "입력하시려는 도메인이 http://guitbe.github.io입니까?" 질문
   → "예" 또는 "네" 클릭!

3. http와 https 둘 다 가능
   - AdSense는 자동으로 https로 변환

───────────────────────────────────────────────────────────────

✅ 체크리스트
───────────────────────────────────────────────────────────────
[ ] 웹사이트 주소 확인 (하위 경로 없는지)
[ ] 최상위 도메인만 AdSense에 입력
[ ] AdSense 질문에 "예" 클릭
[ ] 승인 대기 (1-2주)
