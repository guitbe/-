# 📖 연봉 계산기 웹사이트 사용 가이드

## 🚀 웹사이트 사용 방법

### 1단계: 파일 준비
현재 폴더에 다음 3개의 파일이 있어야 합니다:
- `index.html` (메인 웹페이지)
- `styles.css` (디자인 파일)
- `script.js` (계산 기능 파일)

### 2단계: 로컬에서 테스트하기
1. `index.html` 파일을 더블클릭하거나
2. 브라우저(Chrome, Edge 등)에서 파일을 열기
3. 바로 사용 가능합니다!

### 3단계: 인터넷에 올리기 (필수!)

**옵션 1: 무료 호스팅 서비스 사용 (추천)**
- **GitHub Pages** (무료, 추천!)
  1. GitHub.com 가입
  2. 새 저장소(repository) 만들기
  3. 파일 업로드
  4. Settings → Pages → Source 선택
  5. 몇 분 후 `https://사용자명.github.io/저장소명` 주소로 접속 가능!

- **Netlify** (무료, 매우 쉬움)
  1. Netlify.com 가입
  2. "Add new site" → "Deploy manually"
  3. 파일 드래그 앤 드롭
  4. 즉시 주소 발급!

- **Vercel** (무료, 빠름)
  1. Vercel.com 가입
  2. 프로젝트 업로드
  3. 자동으로 주소 발급!

**옵션 2: 유료 호스팅**
- 카페24, 가비아 등 한국 호스팅 업체
- 월 1,000원~5,000원 정도

## 💰 Google AdSense 수익화 가이드

### Step 1: AdSense 계정 만들기
1. **https://www.google.com/adsense** 접속
2. Google 계정으로 로그인
3. "시작하기" 클릭
4. 웹사이트 URL 입력 (위에서 만든 주소)
5. 개인정보 입력 및 약관 동의

### Step 2: AdSense 승인 받기 (중요!)
- **보통 1-2주 소요**
- 승인 조건:
  ✅ 웹사이트가 실제로 인터넷에 올라가 있어야 함
  ✅ 충분한 콘텐츠 (최소 10-20개 페이지 권장)
  ✅ 트래픽 (일일 방문자 수십 명 이상)
  ✅ 원본 콘텐츠 (복사한 내용 금지)

### Step 3: AdSense 코드 받기
1. AdSense 승인 후
2. "광고" → "광고 단위" 메뉴
3. 새 광고 단위 만들기
4. **발행자 ID** 복사 (ca-pub-로 시작)
5. **광고 슬롯 ID** 복사

### Step 4: 웹사이트에 광고 넣기

**`index.html` 파일 수정:**

1. **9번째 줄 근처** (head 태그 안):
```html
<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-여기에본인발행자ID넣기"
     crossorigin="anonymous"></script>
```
→ `ca-pub-여기에본인발행자ID넣기` 부분을 본인의 발행자 ID로 교체

2. **광고 표시 위치** (이미 추가되어 있음):
- 상단 광고 (헤더 아래)
- 중간 광고 (콘텐츠 사이)
- 하단 광고 (푸터 위) - 원하면 추가 가능

각 광고 위치에서:
- `ca-pub-여기에본인발행자ID넣기` → 본인 발행자 ID
- `data-ad-slot="여기에광고슬롯ID넣기"` → 본인 광고 슬롯 ID

### Step 5: 광고 확인
1. 파일 저장 후 웹사이트 새로고침
2. 광고가 표시되는지 확인
3. AdSense 대시보드에서 "광고 표시" 확인

## 📈 수익화 전략 (체류 시간 늘리기)

### 1. 콘텐츠 추가하기
**추가 페이지 만들기:**
- `guide.html` - 급여 계산 가이드
- `faq.html` - 자주 묻는 질문
- `blog.html` - 급여 관련 블로그
- 각 계산기별 개별 페이지

**방법:**
1. `index.html` 복사해서 `guide.html` 만들기
2. 내용 수정
3. 메뉴에 링크 추가

### 2. SEO 최적화 (검색 엔진 노출)
- 제목에 키워드 포함
- 설명(description) 작성
- 이미지 alt 태그 추가
- 내부 링크 연결

### 3. 소셜 미디어 공유
- 카카오톡, 페이스북 공유 버튼 추가
- 사람들이 공유하면 트래픽 증가

### 4. 정기 업데이트
- 매주 새로운 콘텐츠 추가
- 최신 세율 정보 업데이트
- 사용자 피드백 반영

## 💡 수익화 팁

### AdSense 수익 늘리기
1. **광고 위치 최적화**
   - 콘텐츠 사이에 자연스럽게 배치
   - 너무 많으면 안 좋음 (3-5개 권장)

2. **트래픽 늘리기**
   - 네이버 블로그에 소개
   - 카페, 커뮤니티에 홍보
   - 구글 검색 최적화

3. **사용자 체류 시간 늘리기**
   - 여러 계산기 제공 (이미 구현됨!)
   - 관련 정보 제공
   - 저장 기능 활용

4. **모바일 최적화**
   - 이미 반응형으로 만들어짐
   - 모바일 사용자도 편리하게 사용 가능

### 예상 수익
- **일 방문자 100명**: 월 1만원~5만원
- **일 방문자 1,000명**: 월 10만원~50만원
- **일 방문자 10,000명**: 월 100만원~500만원

*실제 수익은 산업, 지역, 광고 클릭률에 따라 다릅니다*

## 🔧 문제 해결

### 광고가 안 보여요
1. AdSense 승인 받았는지 확인
2. 발행자 ID가 올바른지 확인
3. 브라우저 캐시 삭제 후 새로고침
4. AdSense 대시보드에서 광고 단위 상태 확인

### 계산이 안 돼요
1. `script.js` 파일이 같은 폴더에 있는지 확인
2. 브라우저 개발자 도구(F12)에서 오류 확인
3. 숫자 입력 형식 확인 (예: 50000000)

### 디자인이 깨져요
1. `styles.css` 파일이 같은 폴더에 있는지 확인
2. 파일 이름 정확히 확인 (대소문자 구분)
