  GitHub에 올리는 가장 쉬운 방법 (2분이면 끝!)
═══════════════════════════════════════════════════════════════

방법 1: GitHub 웹사이트에서 직접 업로드 (가장 쉬움!)
───────────────────────────────────────────────────────────────

Step 1: GitHub 저장소 열기
1. https://github.com 접속
2. 로그인
3. 저장소 클릭 (예: guitbe.github.io 또는 salary-calculator)

Step 2: 파일 수정하기
1. 저장소 페이지에서 "index.html" 파일 클릭
2. 우측 상단 "연필 아이콘" (✏️) 클릭 (Edit this file)
3. 파일 내용에서 찾기 (Ctrl + F):
   "ca-pub-여기에본인발행자ID넣기"
4. 모두 다음으로 교체:
   "ca-pub-1741364120484421"
   (총 3곳 교체)
5. 아래쪽 "Commit changes" 버튼 클릭
6. 완료! (자동으로 업데이트됨)

───────────────────────────────────────────────────────────────

방법 2: Netlify 사용 (더 쉬움, 추천!)
───────────────────────────────────────────────────────────────

Step 1: Netlify 접속
1. https://www.netlify.com 접속
2. 로그인

Step 2: 파일 업로드
1. "Add new site" → "Deploy manually" 클릭
2. 다음 파일들 드래그 앤 드롭:
   - index.html
   - styles.css
   - script.js
3. 자동으로 주소 발급!
4. 완료!

💡 장점:
- Git 설치 불필요
- 드래그 앤 드롭만 하면 끝
- 자동 업데이트
- 더 빠름

───────────────────────────────────────────────────────────────

방법 3: GitHub Desktop 사용 (중급)
───────────────────────────────────────────────────────────────

1. GitHub Desktop 다운로드
   https://desktop.github.com
2. 설치 후 로그인
3. 저장소 열기
4. 파일 수정
5. "Commit to main" 클릭
6. "Push origin" 클릭
7. 완료!
