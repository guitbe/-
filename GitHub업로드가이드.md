# 🚀 GitHub에 웹사이트 올리기 (초보자용)

## 현재 상황
✅ 파일들은 로컬 컴퓨터에만 있습니다  
❌ 아직 인터넷에 올라가지 않았습니다  
→ **직접 GitHub에 업로드해야 AdSense 신청 가능합니다!**

---

## 📋 준비물
1. **GitHub 계정** (무료, 5분이면 만들 수 있음)
2. **인터넷 연결**
3. **현재 폴더의 파일들** (index.html, styles.css, script.js)

---

## 🎯 방법 1: GitHub 웹사이트에서 직접 업로드 (가장 쉬움!)

### Step 1: GitHub 계정 만들기
1. **https://github.com** 접속
2. 우측 상단 "Sign up" 클릭
3. 이메일, 비밀번호 입력
4. 계정 생성 완료!

### Step 2: 새 저장소(Repository) 만들기
1. GitHub 로그인 후
2. 우측 상단 **"+"** 버튼 클릭 → **"New repository"** 선택
3. 저장소 이름 입력 (예: `salary-calculator` 또는 `연봉계산기`)
4. **"Public"** 선택 (무료 호스팅을 위해 필수!)
5. ✅ **"Add a README file"** 체크 해제 (이미 파일이 있으므로)
6. **"Create repository"** 클릭

### Step 3: 파일 업로드하기
1. 생성된 저장소 페이지에서
2. **"uploading an existing file"** 링크 클릭
   (또는 "Add file" → "Upload files" 클릭)
3. 파일 드래그 앤 드롭:
   - `index.html`
   - `styles.css`
   - `script.js`
4. 아래쪽 **"Commit changes"** 버튼 클릭
5. 완료!

### Step 4: GitHub Pages 활성화 (웹사이트 주소 만들기)
1. 저장소 페이지에서
2. 상단 메뉴 **"Settings"** 클릭 (저장소 이름 옆에 있음)
3. 왼쪽 사이드바에서 **"Pages"** 클릭
   - 스크롤해서 찾거나, 왼쪽 메뉴에서 찾기
4. **"Source"** 섹션에서:
   - 첫 번째 드롭다운: **"Deploy from a branch"** 선택
   - 두 번째 드롭다운: **"main"** (또는 "master") 선택
   - 세 번째 드롭다운: **"/ (root)"** 선택
5. **"Save"** 버튼 클릭
6. 잠시 기다리기 (1-2분)
   - 페이지를 새로고침하면 상태가 바뀜
7. 위에 **초록색 박스**에 **웹사이트 주소** 표시됨!
   ```
   예시 주소:
   https://yourname.github.io/salary-calculator
   https://홍길동.github.io/연봉계산기
   ```
   ⚠️ 주의: 주소가 나타나기까지 최대 5분 걸릴 수 있습니다!

### Step 5: 웹사이트 확인하기
1. 위에서 받은 주소를 복사
2. 새 브라우저 탭에서 주소 입력 (또는 주소창에 붙여넣기)
3. 엔터 키 누르기
4. 웹사이트가 보이면 성공! 🎉

**확인 체크리스트:**
- ✅ 웹사이트가 로드되는가?
- ✅ 디자인이 제대로 보이는가?
- ✅ 계산 기능이 작동하는가?
- ✅ 모바일에서도 잘 보이는가?

**만약 안 보인다면:**
- 5분 더 기다려보기 (처음 배포는 시간이 걸림)
- 주소가 정확한지 확인 (대소문자 구분)
- `index.html` 파일이 루트에 있는지 확인

## 🎯 방법 2: Netlify 사용 (더 쉬움, 추천!)

### Step 1: Netlify 가입
1. **https://www.netlify.com** 접속
2. **"Sign up"** 클릭
3. GitHub 계정으로 로그인 (또는 이메일로 가입)

### Step 2: 파일 업로드
1. 대시보드에서 **"Add new site"** → **"Deploy manually"** 클릭
2. 파일 드래그 앤 드롭:
   - `index.html`
   - `styles.css`
   - `script.js`
3. 자동으로 업로드 시작
4. 몇 초 후 **웹사이트 주소 발급!**
   - 예: `https://random-name-123.netlify.app`

### Step 3: 커스텀 도메인 (선택사항)
1. **"Site settings"** → **"Change site name"**
2. 원하는 이름으로 변경
3. 예: `salary-calculator.netlify.app`

---

## 🎯 방법 3: Vercel 사용 (빠름!)

### Step 1: Vercel 가입
1. **https://vercel.com** 접속
2. GitHub 계정으로 로그인

### Step 2: 프로젝트 배포
1. **"Add New Project"** 클릭
2. GitHub 저장소 선택 (또는 파일 업로드)
3. 자동으로 주소 발급!

## ✅ AdSense 신청용 웹사이트 주소

위 방법 중 하나로 웹사이트를 올리면:
- ✅ **실제 인터넷 주소**가 생깁니다
- ✅ **누구나 접속 가능**합니다
- ✅ **AdSense 신청 가능**합니다

**예시 주소:**
- GitHub Pages: `https://yourname.github.io/salary-calculator`
- Netlify: `https://salary-calculator.netlify.app`
- Vercel: `https://salary-calculator.vercel.app`

이 주소를 AdSense 신청할 때 입력하면 됩니다!

## 🔍 확인 방법

웹사이트가 제대로 올라갔는지 확인:
1. 주소를 브라우저에 입력
2. 웹사이트가 보이면 ✅ 성공!
3. 계산 기능이 작동하는지 테스트
   - 연봉 입력하고 "계산하기" 버튼 클릭
   - 결과가 나오는지 확인
4. 모바일에서도 잘 보이는지 확인
   - 스마트폰에서 같은 주소 접속
   - 반응형으로 잘 보이는지 확인

**웹사이트 주소 예시:**
```
✅ 정상: https://yourname.github.io/salary-calculator
❌ 오류: https://github.com/yourname/salary-calculator (이건 저장소 주소)
```

**주소 차이:**
- 저장소 주소: `github.com/사용자명/저장소명` (코드 보는 곳)
- 웹사이트 주소: `사용자명.github.io/저장소명` (실제 웹사이트)

## ⚠️ 주의사항

1. **Public 저장소 필수**
   - GitHub Pages는 Public 저장소만 무료 호스팅 가능
   - Private은 유료 플랜 필요

2. **파일 이름 확인**
   - `index.html` (소문자, 정확히!)
   - `styles.css` (소문자)
   - `script.js` (소문자)

3. **첫 배포는 시간 걸림**
   - GitHub Pages: 1-5분
   - Netlify: 즉시
   - Vercel: 즉시

## 🆘 문제 해결

### Q: 웹사이트 주소가 안 나타나요
A: 
1. Settings → Pages에서 설정이 올바른지 확인
   - Source: "Deploy from a branch"
   - Branch: "main" (또는 "master")
   - Folder: "/ (root)"
2. 파일이 제대로 업로드되었는지 확인
   - 저장소 페이지에서 `index.html` 파일이 보이는지 확인
3. 5-10분 더 기다려보기 (처음 배포는 시간이 걸림)
4. 페이지 새로고침 (F5)

### Q: 웹사이트가 안 보여요 (404 에러)
A:
- 주소가 정확한지 확인
  - ✅ `https://사용자명.github.io/저장소이름`
  - ❌ `https://github.com/사용자명/저장소이름` (이건 저장소 주소)
- `index.html` 파일이 루트에 있는지 확인
  - 저장소 페이지에서 파일이 바로 보여야 함
  - 폴더 안에 있으면 안 됨
- 파일 이름 확인 (대소문자 구분)
  - ✅ `index.html` (소문자)
  - ❌ `Index.html` 또는 `INDEX.HTML`

### Q: 디자인이 깨져요 (CSS가 안 적용됨)
A:
- `styles.css` 파일이 업로드되었는지 확인
- 파일들이 같은 폴더(루트)에 있는지 확인
- 브라우저 캐시 삭제 후 새로고침 (Ctrl + F5)

### Q: 계산이 안 돼요 (JavaScript 오류)
A:
- `script.js` 파일이 업로드되었는지 확인
- 브라우저 개발자 도구 열기 (F12)
- Console 탭에서 오류 메시지 확인
- 파일 이름이 정확한지 확인 (`script.js` 소문자)

### Q: 주소는 나왔는데 "사이트를 빌드하는 중"이라고 나와요
A:
- 정상입니다! 1-5분 기다리면 완료됩니다
- 페이지를 새로고침하면 완료 상태로 바뀜
