# 💰 연봉 계산기 웹사이트

연봉 실수령액, 퇴직금, 연차수당 등을 계산하는 웹사이트입니다.

## 📁 파일 구조

```
새 폴더/
├── index.html      # 메인 웹페이지
├── styles.css      # 디자인 스타일
├── script.js       # 계산 로직
├── 사용가이드.md   # 상세 사용 가이드
└── README.md       # 이 파일
```

## 🚀 빠른 시작

1. **로컬에서 테스트**
   - `index.html` 파일을 브라우저에서 열기

2. **인터넷에 올리기**
   - GitHub Pages, Netlify, Vercel 등 무료 호스팅 사용
   - 자세한 방법은 `사용가이드.md` 참고

3. **AdSense 수익화**
   - Google AdSense 계정 만들기
   - `index.html`에 발행자 ID 입력
   - 자세한 방법은 `사용가이드.md` 참고

## ✨ 주요 기능

- ✅ 연봉 실수령액 계산
- ✅ 퇴직금 계산
- ✅ 연차수당 계산
- ✅ 세금 상세 분석
- ✅ 연봉 비교
- ✅ 계산 결과 저장
- ✅ 결과 공유
- ✅ 반응형 디자인 (모바일 지원)

## 📖 상세 가이드

자세한 사용 방법과 수익화 전략은 **`사용가이드.md`** 파일을 참고하세요!

## 🔧 AdSense 설정

`index.html` 파일에서 다음 부분을 찾아 본인의 AdSense 정보로 교체하세요:

1. **발행자 ID**: `ca-pub-여기에본인발행자ID넣기`
2. **광고 슬롯 ID**: `data-ad-slot="여기에광고슬롯ID넣기"`

총 3곳을 수정해야 합니다 (head 태그 1곳, 광고 영역 2곳).
